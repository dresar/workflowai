# 3. Fitur Utama

Berikut adalah modul dan fitur utama yang ada di AI Workflow Studio:

## 1. Ideation & PRD Generator
Sistem mampu mengubah ide mentah dari input pengguna menjadi Product Requirements Document (PRD) yang terstruktur, lengkap dengan target audiens, fitur MVP, dan batasan teknis.

## 2. Diagram & Flowchart Builder
Mendukung pembuatan diagram alur kerja (workflow) secara instan. Representasi visual ini membantu pengembang dan AI memahami logika navigasi dan status aplikasi.

## 3. Tech Stack Suggester
Otomatis memberikan rekomendasi tumpukan teknologi (misalnya: Next.js + Tailwind + Supabase) berdasarkan kebutuhan yang tertuang di PRD.

## 4. Vibe Coding Prompt Compiler
Fitur andalan yang menyatukan PRD, Diagram, dan Tech Stack menjadi satu prompt JSON/Markdown raksasa yang tidak akan di-skip oleh agen AI lainnya saat mengeksekusi pembuatan kode aplikasi.
